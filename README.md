# SproutHelm

**The AI-powered chat widget that turns your agency's website into a client pipeline.**

SproutHelm helps home health care agencies grow by letting families browse caregiver profiles and start real conversations — directly from any website. Every chat is a lead.

## What's Here

This repo contains the SproutHelm marketing website — a static site built with plain HTML/CSS/JS.

- `index.html` — Main landing page
- `blog/` — SEO content articles
- `images/` — Feature illustrations and hero images
- `sprouthelm_logo.svg` — Brand logo

## Local Development

Just open `index.html` in a browser. No build step, no dependencies.

```bash
# Or use a simple local server
python3 -m http.server 8080
```

## Deploy

This is a static site. Deploy anywhere:
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
- Any static hosting
